# Tlias 教学管理系统

## 项目简介

本项目 Tlias 教学管理系统是黑马程序员 JavaWeb + AI 课程的实战项目的后端部分。该系统涵盖了班级管理、学员管理、部门管理、员工管理等功能，帮助学校和教师更高效地管理教学资源和学生信息。

## 技术栈

- 后端：Java、Spring Boot、MyBatis
- 数据库：MySQL
